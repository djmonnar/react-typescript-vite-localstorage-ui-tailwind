# 창작 종족 전투 실험실

React, TypeScript, Vite, Tailwind CSS, localStorage로 만든 모바일 우선 창작 종족 전투 시뮬레이터입니다.

## Features

- 종족, 유닛, 공격 타입, 방어 타입, 종족 특성 CRUD
- 공격 타입 x 방어 타입 상성표 편집
- 자동 전투 로그 시뮬레이션
- 1회 전투 및 100회 반복 결과
- localStorage 저장
- JSON 내보내기/가져오기

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
